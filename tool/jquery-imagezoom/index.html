$('#zoom li').imagezoom({
	'hoverEvent':true
});

调用方法：

$(selector).slider(options);

options参数

参数 默认值 说明
width [计算] 图片外层宽度
height [计算] 图片外层高度
resizeable true 窗口大小改变时是否重新调整图片位置
effect 'out' 图片居中方式处理
condition 'img' 默认筛选条件
hoverEvent false 鼠标悬浮时是否放大
hoverRatio 1.2 鼠标悬浮时放大比例
duration 300 鼠标悬浮时放大动画时长
